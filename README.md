# Candlestick EA

Candlestick-based Expert Advisor for MetaTrader 5.

**Author:** Yousuf Mesalm | www.yousufmesalm.com | WhatsApp +201006179048 | Upwork: https://www.upwork.com/freelancers/youssefmesalm

## Requirements

- MetaTrader 5
- YM library in `MQL5/Include/YM` (if used)

## Installation

Copy `Candle stick EA.mq5` into `MQL5/Experts/`.

## Compilation

Compile in MetaEditor (F7). Set Include path to the terminal MQL5 folder.

## Usage

Attach to a chart and set lot size, magic number, and other inputs.
